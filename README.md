# vue-phone-preview

## 适用于pc端后台管理系统手机预览

> A VueCli3.x+TypeScript Toolkit for Web.

## Install
```
$ npm i --save vue-phone-preview
or
$ yarn add vue-phone-preview
```

## Quick Start
``` javascript
import Vue from 'vue'
import VuePhonePreview from 'vue-phone-preview';

Vue.use(VuePhonePreview)
@Component()
// or
import VuePhonePreview from 'vue-phone-preview'

@Component({
  components: {
    VuePhonePreview
  }
})
```
